data-pid="x-xQpqlc">咳咳，为什么可以这样口出狂言呢？这得从逻辑语的基本句法说起。在一个简单的陈述句中，句子一般由一个谓语和1~5个“位”组成，“位”的概念类似于函数的参数，根据谓词的不同，位的数量上限和每个位分别表达的含义都不同。将谓语和不同的位按一定顺序组合，就有可能直接表达一个比较复杂的含义。</p><p data-pid="QxeLc3Ba">在这里举个例子：</p><blockquote data-pid="kifIPv5c"><i>la</i> lis.xuas. <i>cu</i> <b>fanva</b> <i>la</i> das.kapital. <i>le</i> jungo <i>le</i> dotco <i>la</i> dzbynluyn.</blockquote><p data-pid="Ybv_reEW">这句话中，粗体的 fanva 作谓语，意思是“ x_1 翻译文本/话语 x_2 至目标语言 x_3 ，来自源语言 x_4 ，得到翻译结果 x_5 ”（ x 表示不同的位）；斜体的 la 和 le 用作冠词， cu 分隔位和谓语，除此之外句子中没有任何虚词。而这句话直译成中文是个什么样子呢</p><blockquote data-pid="0LshNLUd">李华 - <b>翻译 </b>- Das Kapital - 中国的 - 德国的 - 资本论</blockquote><p data-pid="0oQAzUvj">说成人话就是</p><blockquote data-pid="WWDbEQs2">李华<i>把</i>德文<i>的</i> Das Kapital <b>翻译</b><i>成</i>中文<i>的</i>《资本论》。</blockquote><p data-pid="AI40PfSi">看到没有？中文的说法平白增添了许多介词和助词，反而不利于语序的改变。而在逻辑语当中，这些句子成分可以相当随意地调换。首先来个基础的，句子中的 cu fanva 可以移到任意一个位后面：</p><blockquote data-pid="27wgwI-T">la lis.xuas. <b>cu fanva </b>la das.kapital. le jungo le dotco la dzbynluyn.<br/>la lis.xuas. la das.kapital. <b>cu fanva</b> le jungo le dotco la dzbynluyn.<br/>la lis.xuas. la das.kapital. le jungo <b>cu fanva</b> le dotco la dzbynluyn.<br/>la lis.xuas. la das.kapital. le jungo le dotco <b>cu fanva</b> la dzbynluyn.<br/>la lis.xuas. la das.kapital. le jungo le dotco la dzbynluyn. <b>cu fanva</b></blockquote><p data-pid="hJcuuiZn">以上五个句子的意思完全一致，且都没有语法错误。</p><figure data-size="normal"><noscript><img src="https://pic1.zhimg.com/v2-52ce004a27e1a1197443b3091c27fd58_b.jpg" data-caption="" data-size="normal" data-rawwidth="161" data-rawheight="187" class="content_image" width="161"/></noscript><img src="data:image/svg+xml;utf8,&lt;svg xmlns=&#39;http://www.w3.org/2000/svg&#39; width=&#39;161&#39; height=&#39;187&#39;&gt;&lt;/svg&gt;" data-caption="" data-size="normal" data-rawwidth="161" data-rawheight="187" class="content_image lazy" width="161" data-actualsrc="https://pic1.zhimg.com/v2-52ce004a27e1a1197443b3091c27fd58_b.jpg"/></figure><p data-pid="dLcUFdCk">嗯，这个当然可以，利用虚词 se 、 te 、 ve 、 xe ，我们可以把句子中的第一个位和第二/三/四/五个位交换位置，由此我们可以得到如下的句子</p><blockquote data-pid="KnwWs8cU">la lis.xuas. cu fanva la das.kapital. le jungo le dotco la dzbynluyn.<br/>（李华 - 翻译- Das Kapital - 中国的 - 德国的 - 资本论）<br/><i><b>la das.kapital.</b></i> cu <b>se</b> fanva <i><b>la lis.xuas.</b></i> le jungo le dotco la dzbynluyn.<br/>（<b>Das Kapital</b> - 翻译 - <b>李华</b> - 中国的 - 德国的 - 资本论）<br/><i><b>le jungo</b></i> cu <b>te</b> fanva la das.kapital. <i><b>la lis.xuas.</b></i> le dotco la dzbynluyn.<br/>（<b>中国的</b> - 翻译 - Das Kapital - <b>李华</b> - 德国的 - 资本论）<br/><i><b>le dotco</b></i> cu <b>ve</b> fanva la das.kapital. le dotco <i><b>la lis.xuas. </b></i>la dzbynluyn.<br/>（<b>德国的</b> - 翻译 - Das Kapital - 中国的 - <b>李华</b> - 资本论）<br/><i><b>la dzbynluyn</b></i> cu <b>xe</b> fanva la das.kapital. le jungo le dotco <i><b>la lis.xuas.</b></i><br/>（<b>资本论</b> - 翻译 - Das Kapital - 中国的 - 德国的 - <b>李华</b>）</blockquote><p data-pid="AeAdRpAf">在这几个句子中，我们可以发现第一个位 la lis.xuas. （李华）分别与其他四个位交换了位置，那么句子的意思有没有发生改变呢？答案当然是没有。</p><p data-pid="0WT39GEN">语序自由到这种程度，估计多数语言的使用者已经觉得能把人逼疯了，精神正常的人都不会再在这上面折腾了。但是——</p><figure data-size="normal"><noscript><img src="https://pic1.zhimg.com/v2-52ce004a27e1a1197443b3091c27fd58_b.jpg" data-caption="" data-size="normal" data-rawwidth="161" data-rawheight="187" class="content_image" width="161"/></noscript><img src="data:image/svg+xml;utf8,&lt;svg xmlns=&#39;http://www.w3.org/2000/svg&#39; width=&#39;161&#39; height=&#39;187&#39;&gt;&lt;/svg&gt;" data-caption="" data-size="normal" data-rawwidth="161" data-rawheight="187" class="content_image lazy" width="161" data-actualsrc="https://pic1.zhimg.com/v2-52ce004a27e1a1197443b3091c27fd58_b.jpg"/></figure><p data-pid="CEtOnJsr">答案是确实可以再给力一点。</p><p data-pid="2Xx8KDVH">逻辑语中有五个词语： fa / fe / fi / fo / fu （刚好是字母 f 和五个元音字母的组合），分别指示自己后面的内容是第一/二/三/四/五个位，在不需要的时候可以省略。</p><p data-pid="y9N5Rikz">由此，</p><blockquote data-pid="Do13Ey_q">la lis.xuas. cu fanva la das.kapital. le dotco le jungo la dzbynluyn.</blockquote><p data-pid="uXLo7kNg">就可以变成</p><blockquote data-pid="eupjBcAH"><b>fa</b> la lis.xuas. cu fanva <b>fe</b> la das.kapital. <b>fi</b> le jungo <b>fo</b> le dotco <b>fu</b> la dzbynluyn.</blockquote><p data-pid="KjNRLySD">当然这五个词全部可以省略。</p><p data-pid="6frQ7Sei">但只要有了它们，这句话的语序就更加为所欲为了——</p><p data-pid="yF-UuaFP">（下文中[]内的词语可以省略）</p><p data-pid="3_c4oBsa">你可以把这句话正着说：</p><blockquote data-pid="wtStGPu3">[<b>fa</b>] la lis.xuas. cu fanva [<b>fe</b>] la das.kapital. [<b>fi</b>] le jungo [<b>fo</b>] le dotco [<b>fu</b>] la dzbynluyn.<br/>（李华 - 翻译 - Das Kapital - 中国的 - 德国的 - 资本论）</blockquote><p data-pid="pp5KsQ0P">反着说：</p><blockquote data-pid="vOW9zMBq"><b>fu</b> la dzbynluyn. cu fanva <b>fo</b> le dotco <b>fi</b> le jungo <b>fe</b> la das.kapital. <b>fa</b> lis.xuas.<br/>（资本论 - 翻译 - 德国的 - 中国的 - Das Kapital - 李华）</blockquote><p data-pid="Mh9VCM1u">按实词和名字的字母顺序说：</p><blockquote data-pid="9-9LBjgn"><b>fe</b> la das.kapital. cu fanva <b>fo</b> le dotco <b>fu</b> la dzbynluyn. <b>fi</b> le jungo <b>fa</b> lis.xuas.<br/>（Das Kapital - 翻译 - 德国的 - 资本论 - 中国的 - 李华）</blockquote><p data-pid="TcYFu0EE">为了狭隘的民族主义/爱国主义把德国佬的东西放在后边说：</p><blockquote data-pid="XFzPVw-F">[<b>fa</b>] la lis.xuas. cu fanva <b>fi</b> le jungo <b>fu</b> la dzbynluyn. [<b>fo</b>] le dotco <b>fe</b> la das.kapital.<br/>（李华 - 翻译 - 中国的 - 资本论 - 德国的 - Das Kapital）</blockquote><p data-pid="91ZQagnA">甚至，你还可以结合之前提到的“句子中的 cu fanva 可以移到任意一个位后面”，将句子的语序转化成和其他语言一样：</p><blockquote data-pid="FoESH4Ll">李华把德文的 Das Kapital 翻译成中文的《资本论》。<br/>→ 李华 - 德国的 - Das Kapital - 翻译 - 中国的 - 资本论<br/>→ [<b>fa</b>] la lis.xuas. <b>fo</b> dotco <b>fe</b> la das.kapital. cu fanva <b>fi</b> le jungo [<b>fu</b>] la dzbynluyn.</blockquote><p data-pid="N3Yb0VWO">以上所述的所有句子，均与例句有着相同的含义。而这，恐怕已经十分接近人类语言语序自由的极限了吧。</p><p data-pid="2W9igBYa">嗯，真是不知道这么说话的人会不会更容易招人打呢。</p><hr/><p data-pid="zrmmUOZ2">今天就先更到这里吧，码了这么多字也不知道有没有人看，但只要有一个人会要求，我就会想办法继续更新。</p><p comma）。大部分报纸或杂志会省略这个逗号，但有的时候会因此引起歧义，如：</p><p class="ztext-empty-paragraph"><br/></p><p data-pid="rYn2JvE3">We had coffee, cheese and crackers and grapes.</p><p
